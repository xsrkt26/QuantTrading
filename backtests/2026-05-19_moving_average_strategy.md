# Moving Average Strategy Backtest

日期：2026-05-19

## 策略版本

- 策略：双均线趋势过滤策略
- 标的：SPY
- 周期：日线
- 短期均线：20 日
- 长期均线：100 日
- 方向：只做多，不做空
- 仓位：满仓或空仓
- 交易成本：单边 1 bps

## 数据设置

- 数据来源：Yahoo Finance，经 `yfinance` 下载
- 起始日期：2000-01-01
- 实际数据区间：2000-01-03 至 2026-05-18
- 数据行数：6633
- 价格：复权价格，`auto_adjust=True`

## 交易规则

```text
如果 20 日均线 > 100 日均线，产生持有信号。
如果 20 日均线 <= 100 日均线，产生空仓信号。
今天收盘后生成的信号，下一交易日执行。
```

核心防未来函数代码：

```python
position = signal.shift(1)
```

## 回测结果

| 指标 | 买入持有 | 双均线策略 |
| --- | ---: | ---: |
| 最终净值 | 8.0844 | 4.4067 |
| 总收益 | 708.44% | 340.67% |
| 年化收益 | 8.27% | 5.80% |
| 最大回撤 | -55.19% | -24.29% |

其他指标：

- 交易次数：71
- 在场时间：70.32%

## 初步结论

这个 20/100 双均线策略没有跑赢 SPY 买入持有，但显著降低了最大回撤。

这说明它更像是一个趋势过滤和风险控制策略，而不是收益增强策略。它在下跌阶段可能减少损失，但也会在快速反弹或长期牛市中错过部分收益。

## 主要风险

- 参数未经优化，20/100 只是教学参数。
- 没有做样本内和样本外拆分。
- 没有做参数敏感性分析。
- 交易成本只用固定 bps 简化，没有考虑冲击成本。
- 只测试 SPY，不能说明对其他资产有效。
