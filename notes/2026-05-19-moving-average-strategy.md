# Moving Average Strategy

日期：2026-05-19

## 本课目标

从买入持有升级到第一个交易策略：双均线策略。

核心链条：

```text
Close -> ma_short / ma_long -> signal -> position -> strategy_return -> strategy_equity
```

## 均线

均线是最近一段时间价格的平均值。

```text
20 日均线 = 最近 20 个交易日收盘价平均值
100 日均线 = 最近 100 个交易日收盘价平均值
```

短均线反应更快，长均线更平滑。

## 策略规则

- 当 20 日均线高于 100 日均线，认为趋势偏强，持有 SPY。
- 当 20 日均线低于或等于 100 日均线，认为趋势偏弱，空仓。
- 本策略只做多，不做空。

## 核心字段

- `ma_short`：短期均线
- `ma_long`：长期均线
- `signal`：今天收盘后产生的信号
- `position`：今天实际持仓
- `trade`：仓位变化，1 表示买入或卖出一次
- `strategy_return`：策略日收益率
- `strategy_equity`：策略净值曲线
- `strategy_drawdown`：策略回撤曲线

## 为什么要 shift(1)

今天的收盘价只有收盘后才知道，今天的均线信号也只有收盘后才知道。

所以今天产生的信号，最早只能明天执行。

```python
position = signal.shift(1)
```

如果不做 `shift(1)`，就是假装自己在今天交易前已经知道今天收盘价，这属于未来函数。

## 交易成本

本课加入一个简化交易成本：

```text
单边交易成本 = 1 bps = 0.01%
```

如果今天发生一次买入或卖出，就从当天策略收益里扣除这部分成本。
