# Core Quant Principles

日期：2026-05-19

这份笔记记录量化学习初期必须建立的市场观和研究纪律。它和具体交易概念一样重要，甚至更重要。代码能让回测跑起来，但这些观念决定我们是否会被漂亮曲线骗。

## 1. 市场很强，先默认自己没有优势

有效市场的核心含义是：市场价格已经快速反映了大量已知信息。

这不代表市场永远正确，也不代表没有人能赚钱。它真正提醒我们：

```text
想长期、稳定、扣除成本后跑赢市场，很难。
```

每看到一个策略，都先问：

- 为什么这个机会没有被别人赚走？
- 它是不是只是承担了更多风险？
- 它是不是只适合某段历史？
- 扣除手续费、滑点、冲击成本后还存在吗？
- 它的容量适合个人还是机构？

## 2. Alpha 和 Beta 要分清

`Beta` 是市场本身给你的收益。

例如长期持有 SPY，主要赚的是美股市场整体上涨的钱。

`Alpha` 是在市场暴露之外获得的超额收益，或者更好的风险调整收益。

很多策略看起来有 alpha，其实只是暴露在某种 beta 上：

- 股票市场 beta
- 小盘 beta
- 价值 beta
- 动量 beta
- 利率 beta
- 波动率 beta

所以不能只问策略赚不赚钱，还要问：

```text
它到底赚的是市场的钱，还是模型的钱？
```

## 3. 收益不是唯一目标，风险收益比才重要

新手容易只看收益：

```text
这个策略赚了多少？
```

真正应该问：

```text
为了赚这些钱，过程中承受了多大风险？
我能不能活到策略兑现的那一天？
```

例如：

| 策略 | 年化收益 | 最大回撤 |
| --- | ---: | ---: |
| A | 20% | -70% |
| B | 10% | -15% |

A 的收益更高，但从 100 万跌到 30 万，大多数人无法坚持，也可能被迫停止。B 收益较低，但更容易长期执行。

量化评价策略通常一起看：

- 年化收益：赚得快不快
- 年化波动率：净值上下波动有多大
- 最大回撤：历史最惨从高点跌了多少
- 夏普比率：每承担一单位波动，获得多少超额收益
- Calmar 比率：年化收益 / 最大回撤绝对值

亏损和上涨不是对称的：

| 亏损 | 回本需要上涨 |
| ---: | ---: |
| -50% | +100% |
| -70% | +233% |
| -90% | +900% |

所以量化追求的不是最高收益，而是：

```text
可承受回撤下的长期复利。
```

## 4. 回测不是赚钱证明，只是假设检验

回测结果好，只能说明：

```text
这个规则在这段历史数据上表现不错。
```

它不能证明未来一定赚钱。

回测的正确用途是检验假设：

- 这个规则是否有基本逻辑？
- 历史上是否表现过？
- 成本后是否还能活？
- 参数是否稳定？
- 换一个时间段是否还可以？

## 5. 未来函数是第一大毒药

未来函数指使用了当时不可能知道的信息。

例如今天收盘后才知道今天的均线信号，却用这个信号赚今天的钱，就是偷看未来。

错误方式：

```python
position = signal
```

更合理的方式：

```python
position = signal.shift(1)
```

这表示今天收盘后产生的信号，下一交易日才执行。

只要存在未来函数，回测结果基本作废。

## 6. 过拟合比亏钱更危险

过拟合是指策略没有抓住真实规律，而是记住了历史数据里的噪声。

例如不断试参数：

```text
5/20 不好
10/50 不好
17/113 很好
23/91 更好
31/147 历史最强
```

最后选出来的“历史最强参数”，可能只是刚好适配过去那段行情。未来环境一变，它就失效。

过拟合危险在于：它会给你一条很漂亮的历史净值曲线。

亏钱策略一眼能看出来不好，过拟合策略更危险，因为它让你误以为自己找到了规律。

常见过拟合行为：

- 反复调参数直到回测好看
- 加很多过滤条件
- 试了几十个策略只保留最好的
- 只展示表现最好的时间段
- 忽略交易成本
- 没有样本外验证

过拟合策略的典型特征：

- 参数稍微一变，收益就崩
- 交易次数很少，但收益特别夸张
- 逻辑解释牵强
- 样本内很好，样本外很差
- 扣除成本后收益消失
- 规则越来越复杂

防过拟合方法：

- 规则尽量简单
- 参数不要追求历史最优
- 看参数附近一片区域是否都有效
- 做样本外测试
- 做交易成本压力测试
- 记录失败实验，不只记录成功实验

重要原则：

```text
历史最优参数通常不是最可靠参数。
```

更可靠的策略通常不是某个参数点特别强，而是一片参数区域都表现尚可。

## 7. 交易成本会吃掉很多纸面 alpha

交易成本包括：

- 手续费
- 滑点
- 买卖价差
- 冲击成本
- 税费

高换手策略尤其危险：

```text
毛收益看起来不错，净收益可能为负。
```

每个策略都必须问：

- 一年交易几次？
- 换手率多少？
- 成本翻倍后还赚钱吗？
- 在流动性变差时还能成交吗？

## 8. 容量决定策略适合谁

容量是一个策略能容纳多少资金。

有些策略对大机构太小，但对个人正合适。

小资金优势：

- 冲击成本小
- 进出更灵活
- 可以做机构嫌太小的机会
- 不必把策略产品化给客户解释

但小容量机会通常也更脆弱、更难执行。

个人量化不要和机构正面对抗。更现实的方向是：

```text
避开机构主战场，用小资金灵活性、低容量机会、长期纪律和严格风控寻找小 alpha。
```

## 9. 市场状态会变化

一个策略可能在趋势市场有效，在震荡市场失效。

不要只问：

```text
这个策略赚不赚钱？
```

还要问：

```text
它在什么市场环境赚钱？
它在什么市场环境亏钱？
现在是否还是那个环境？
```

双均线策略就是典型例子：

- 趋势市场可能表现好
- 震荡市场容易反复买卖
- 快速反弹时可能错过部分上涨

## 10. 分散化不是买很多东西，而是分散风险来源

买 20 只科技股，不一定是真分散。

真正的分散是不同收益来源：

- 股票 beta
- 债券 beta
- 商品趋势
- 横截面动量
- 价值
- 低波
- 现金或防守资产

相关性在危机时可能升高，所以分散化也需要压力测试。

## 11. 仓位管理常常比买卖信号更重要

同一个交易信号，用不同仓位规则，结果会完全不同：

- 满仓或空仓
- 固定半仓
- 波动率越大仓位越小
- 最大回撤超过阈值就降仓
- 单资产最大仓位限制

量化不是只研究：

```text
买什么？
```

还要研究：

```text
买多少？
亏到多少减仓？
什么时候停止？
```

## 12. 策略必须有基准

没有基准，就不知道策略是否真的有价值。

例如 SPY 策略至少要和：

- SPY 买入持有
- 现金
- 简单双均线策略
- 简单动量策略

进行比较。

如果复杂模型跑不过简单基准，复杂就没有意义。

## 13. 简单策略先过关，再谈复杂模型

机器学习和深度学习不是捷径。

如果复杂模型不能稳定跑赢简单均线、简单动量或等权组合，那复杂只是增加了过拟合风险。

学习顺序应该是：

```text
简单规则 -> 稳健验证 -> 组合和风控 -> 再考虑复杂模型
```

## 14. 样本外比样本内重要

样本内是用来研究、调参、选择策略的历史数据。

样本外是不能反复修改、不能用来调参的检验区间。

时间序列不能随机打乱，必须按时间切。

例如 SPY 数据区间是：

```text
2000-01-03 到 2026-05-18
```

可以划分为：

```text
训练区间：2000-01-03 到 2014-12-31
验证区间：2015-01-01 到 2019-12-31
测试区间：2020-01-01 到 2026-05-18
```

用途：

- 训练区间：想策略、试参数
- 验证区间：比较少数候选参数
- 测试区间：最终只跑一次，作为真实检验

代码示意：

```python
train = strategy.loc["2000-01-01":"2014-12-31"]
valid = strategy.loc["2015-01-01":"2019-12-31"]
test = strategy.loc["2020-01-01":]
```

最重要的纪律：

```text
样本外不能反复用来调参。
```

如果看了 2020-2026 表现不好，然后又改参数让 2020-2026 好看，这段测试集就被污染了。它已经变成新的样本内。

更专业的方法是 walk-forward：

```text
用 2000-2009 训练，2010-2011 测试
用 2002-2011 训练，2012-2013 测试
用 2004-2013 训练，2014-2015 测试
不断向前滚动
```

## 15. 策略不是预测未来，而是管理概率

量化不是算出：

```text
明天一定涨。
```

更现实的目标是：

```text
长期重复做一类正期望决策。
```

单次交易可以亏，短期可以回撤，但长期如果逻辑、数据、成本和风控都站得住，策略才有研究价值。

## 每个策略的观念检查清单

每次研究策略，都至少问一遍：

- 这个策略赚的是 alpha 还是 beta？
- 它和什么基准比较？
- 它的最大回撤我能否承受？
- 它是否使用了未来数据？
- 它是否可能只是过拟合？
- 参数附近是否也有效？
- 扣除成本后是否仍然有效？
- 成本翻倍后是否仍然能活？
- 样本外表现如何？
- 它在哪种市场环境下失效？
- 它的容量是否适合个人资金？
- 如果连续亏损，我是否知道何时停用？

## 当前牢记

```text
市场很强，回测会骗人，成本很真实，风险比收益更重要，样本外才是真考试。
```
