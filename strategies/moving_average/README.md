# Moving Average Strategy

## 策略假设

市场存在中长期趋势。当短期均线高于长期均线时，说明近期价格强于长期趋势，继续持有可能获得趋势收益；当短期均线低于长期均线时，说明趋势走弱，空仓可以降低下跌阶段的损失。

## 交易标的和周期

- 标的：SPY
- 周期：日线
- 数据起始：2000-01-01
- 价格：复权价格

## 策略规则

- 短期均线：20 日均线
- 长期均线：100 日均线
- `ma_short > ma_long`：持有 SPY
- `ma_short <= ma_long`：空仓
- 只做多，不做空

## 防止未来函数

今天收盘后才能计算今天的均线信号，所以今天的信号不能用于今天的收益。

实际持仓使用：

```python
position = signal.shift(1)
```

这表示昨天生成的信号，决定今天的仓位。

## 交易成本

本课默认单边交易成本为 `1 bps`，即 `0.01%`。

仓位从 0 到 1 或从 1 到 0 时，视为发生一次交易，并扣除交易成本。

## 主要风险

- 均线策略在震荡市场容易频繁买卖。
- 参数选择可能过拟合。
- 本策略只用日线收盘价，没有考虑盘中成交价格。
- 本策略没有考虑税费、冲击成本和真实下单延迟。
- 历史趋势不保证未来继续有效。

## 失效条件

- 市场长期横盘震荡，均线频繁交叉。
- 趋势突然反转，均线反应滞后。
- 交易成本上升导致小趋势被成本吃掉。
- 参数只在特定历史区间有效，样本外表现明显变差。
