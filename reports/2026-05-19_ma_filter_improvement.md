# Moving Average Filter Improvement Report

日期：2026-05-19

## 研究目的

第 7 课发现 SPY `10/200` 双均线策略的主要收益来自长期趋势交易，短期交易整体亏损。本报告测试几种简单过滤器，观察它们是否能减少短期假突破，同时保留长期趋势收益。

## 策略设置

- 标的：SPY
- 数据源：Yahoo Finance，经 `yfinance` 下载
- 数据区间：2000-01-03 至 2026-05-18
- 基准策略：10/200 双均线
- 执行模型：next-open
- 滑点：单边 2 bps
- 佣金：单边 1 bps

## 过滤器定义

| variant | 规则 |
| --- | --- |
| baseline | 原始 10/200 双均线 |
| confirm_3d | 连续 3 天满足信号才改变持仓状态 |
| confirm_5d | 连续 5 天满足信号才改变持仓状态 |
| band_1pct | 短均线高于长均线 1% 才买入，低于长均线 1% 才卖出 |
| vol_cap_25pct | 20 日年化波动率低于 25% 才允许持有 |

## 实验结果

## 观察

- `confirm_3d` 和 `confirm_5d` 降低了短期交易拖累，但最大回撤明显变差。
- `band_1pct` 明显减少交易次数和短期交易数量，同时略微提高最终交易净值。
- `vol_cap_25pct` 最终净值最高，但交易次数和短期交易数量都增加，不符合“减少假突破”的主要目标。
- 所有版本仍然具有收益集中现象，说明双均线趋势策略仍依赖少数大趋势交易。

## 结论

从本课目标看，`band_1pct` 是更值得继续研究的改进候选。

但当前结果仍只是一次历史实验。后续必须继续检查：

- 样本内 / 验证 / 样本外表现
- `band_pct` 参数敏感性
- 其他 ETF 或指数是否稳定
- 是否错过关键长期趋势
- 成本和滑点提高后是否仍然有效
