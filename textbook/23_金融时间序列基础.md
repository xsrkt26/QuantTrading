## 详细讲解

### 1. 第 23 章为什么重要

前面我们一直在研究策略和因子，但第 23 章先停下来问一个基础问题：

```text
金融收益数据到底长什么样？
```

这很重要，因为很多错误策略都来自一个误判：

```text
把随机噪声当成稳定规律。
```

如果你不了解收益分布、厚尾、自相关和波动聚集，就很容易被漂亮回测骗到。

### 2. 本章研究的对象

本章只看 SPY 日收益：

```python
returns = close.pct_change().dropna()
```

也就是每天：

```text
今天收盘价 / 昨天收盘价 - 1
```

它不是策略收益，而是最基础的市场收益序列。

学会读市场收益序列，是后面判断策略是否靠谱的基础。

### 3. daily_mean 怎么理解

本章结果：

```text
daily_mean = 0.0005
```

这表示 SPY 在样本期内平均日收益约为 0.05%。

看起来很小，但长期复利后会变成可观收益。

不过你不能因为日均收益为正，就觉得每天上涨概率很稳。日收益的噪声非常大，单日涨跌主要由波动主导。

所以日均收益更适合长期统计，不适合预测明天。

### 4. annualized_volatility 是什么

本章结果：

```text
annualized_volatility = 0.1930
```

也就是年化波动率约 19.30%。

它的计算逻辑是：

```python
annualized_volatility = daily_return_std * sqrt(252)
```

这里 252 是一年大约 252 个交易日。

波动率不是亏损，而是收益上下波动的幅度。波动越高，净值曲线越颠簸，回撤风险通常也更高。

### 5. skewness 是什么

本章结果：

```text
skewness = -0.0018
```

偏度衡量收益分布是否左右不对称。

```text
偏度为正：右尾更长，大涨极端值更多。
偏度为负：左尾更长，大跌极端值更多。
接近 0：整体左右比较接近。
```

本章 SPY 的偏度接近 0，但这不代表风险低，因为还要看厚尾。

### 6. excess_kurtosis 和厚尾

本章结果：

```text
excess_kurtosis = 15.0166
```

这是非常重要的数字。

正态分布的 excess kurtosis 理论上接近 0。这里是 15 左右，说明 SPY 日收益有明显厚尾：

```text
极端大涨和极端大跌，比正态分布假设下更常见。
```

这就是为什么风险管理不能只相信正态分布。

如果你按正态分布估计风险，就会低估极端行情出现的概率。

### 7. best_day 和 worst_day 的意义

本章结果：

```text
best_day = 14.52%
worst_day = -10.94%
```

这说明在样本期里，SPY 出现过非常极端的单日涨跌。

对策略来说，这有两个提醒：

```text
第一，极端日会显著影响回测结果。
第二，策略必须考虑跳空、流动性和无法按理想价格成交的情况。
```

尤其是趋势策略和均值回归策略，在极端日附近可能表现完全不同。

### 8. autocorr 是什么

自相关衡量的是：

```text
今天的收益和过去收益有没有关系。
```

本章代码是：

```python
autocorr = returns.autocorr(lag=1)
```

`lag=1` 就是看：

```text
昨天涨跌和今天涨跌有没有线性关系。
```

本章结果：

```text
autocorr_1d = -0.1039
autocorr_5d = -0.0130
```

1 日自相关略微为负，5 日自相关非常接近 0。

这说明：

```text
单纯用最近一两天涨跌去预测下一天，非常不稳定。
```

这也是为什么我们前面做策略时，不会只看昨天涨了就买、昨天跌了就卖。那太容易把噪声当规律。

### 9. 滚动波动率为什么重要

本章图右侧是 63 日滚动年化波动率：

```python
rolling_vol = returns.rolling(63).std() * np.sqrt(252)
```

它告诉我们：

```text
市场风险不是固定的。
```

有些时期波动很低，有些时期波动突然升高，并且高波动往往会持续一段时间。

这叫波动聚集：

```text
高波动后面容易跟着高波动；
低波动后面容易跟着低波动。
```

第 13 章的目标波动率仓位，就是基于这个事实：市场风险状态会变化，所以仓位也要能调整。

### 10. 为什么不能轻易相信正态假设

很多简单模型会默认：

```text
收益近似正态分布；
每天独立同分布；
波动率相对稳定。
```

第 23 章告诉你，这些假设都要谨慎。

真实金融收益更接近：

```text
均值很小；
噪声很大；
极端值比想象中更多；
波动率会聚集；
短期自相关通常不稳定。
```

所以一个策略如果只在平稳、正态、低波动假设下有效，实盘很可能扛不住。

### 11. 第 23 章和策略开发的关系

这章不是教你一个买卖策略，而是教你判断策略是否现实。

以后看到一个策略，你要问：

```text
它是否只靠少数极端日赚钱？
它在高波动时期会不会崩？
它有没有低估厚尾风险？
它的信号是不是只是短期噪声？
```

这些问题比单纯看最终收益更重要。

真正成熟的量化研究，不是只找漂亮净值曲线，而是理解净值曲线背后的收益分布和风险结构。

### 12. 本章过关标准

你能讲清楚下面四句话，第 23 章就算过关：

```text
金融日收益均值很小，噪声很大，不能轻易预测单日涨跌。
厚尾表示极端行情比正态分布更常见。
自相关弱时，不要把短期涨跌误判成可预测规律。
波动率会聚集，所以风险状态会随时间变化。
```
