# QuantTrading Textbook

这个目录保存课本式讲义。

从后续课程开始，每节课优先生成一份可以直接阅读的主课本：

```text
textbook/NN_topic.md
```

课本需要同时包含：

- 本课要解决的问题
- 必须理解的概念
- 关键代码片段
- 代码运行结果
- 图表和截图
- 结果解读
- 本课结论
- 下一课衔接

图片和图表统一放在：

```text
textbook/assets/NN_topic/
```

代码仍然保留在 `src/`、`scripts/` 和 `notebooks/` 中。课本负责把学习体验串起来，避免只看代码或只看零散笔记。
